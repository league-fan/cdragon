这是一个数据收集项目，主要功能是定期抓取communitydragon.org的最新数据，如PBE服务器的Champion、Skin等相关数据。经过数据清洗、分类后归档。

可用语言

默认 (default)日语 (ja_jp)韩语 (ko_kr)简体中文 (zh_cn)繁体中文 (zh_tw)

资产类型

skin

皮肤相关数据，包括皮肤名称、价格和描述

skinline

皮肤系列相关数据，包括主题和关联皮肤

champion

英雄/角色相关的数据，包括属性、技能和背景故事

universe

宇宙观设定相关数据，包括地区和角色故事

API 端点

以下是可用的API端点，您可以通过这些端点获取相关数据：

获取版本信息 
GET /version.json
获取特定语言的英雄列表 
GET /:language/champion/index.json

例如：/zh_cn/champion/index.json

获取特定语言的英雄详情 
GET /:language/champion/:championAlias.json

例如：/zh_cn/champion/Aatrox.json

获取特定语言的皮肤列表 
GET /:language/skin/index.json

例如：/zh_cn/skin/index.json

获取特定语言的皮肤详情 
GET /:language/skin/:skinId.json

例如：/zh_cn/skin/1.json

获取特定语言的皮肤系列列表 
GET /:language/skinline/index.json

例如：/zh_cn/skinline/index.json

获取特定语言的皮肤系列详情 
GET /:language/skinline/:skinlineId.json

例如：/zh_cn/skinline/1.json

获取特定语言的宇宙内容 
GET /:language/universe/index.json

例如：/zh_cn/universe/index.json

获取特定语言的宇宙内容详情 
GET /:language/universe/:universeId.json

例如：/zh_cn/universe/1.json

数据更新周期

数据会定期从communitydragon.org获取更新，通常在游戏版本更新后的24小时内完成。
